# LSCS_Take_Home_Exam

## Setup
1. Clone the repository
```bash
git clone https://github.com/Patthethicc/LSCS_Take_Home_Exam
cd LSCS_Take_Home_Exam
```
2. Install the dependencies
```bash
npm install
```
3. Create `.env` file
consists of:
```bash
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_DATABASE=lscs_test
```
4. Run nodemon api.js to start the server
```bash
nodemon api.js
```
